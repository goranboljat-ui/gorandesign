# Blockwell Design System

## Core Visual Identity
Blockwell is a modern, polished block-puzzle game. The design should feel premium, clean, and tactile.

## 1. Color Palette
### Primary Brand Colors
- **Primary:** `#4F46E5` (Indigo) - Main UI elements, buttons
- **Secondary:** `#10B981` (Emerald) - Success states, rewards
- **Accent:** `#F59E0B` (Amber) - Highlights, special blocks
- **Background:** `#0F172A` (Slate 900) - Main game background
- **Surface:** `#1E293B` (Slate 800) - Cards, modals
- **Text Primary:** `#F8FAFC` (Slate 50)
- **Text Secondary:** `#94A3B8` (Slate 400)

### Block Color Palette (Classic Theme)
- **Block 1 (Red):** `#EF4444`
- **Block 2 (Blue):** `#3B82F6`
- **Block 3 (Green):** `#10B981`
- **Block 4 (Yellow):** `#F59E0B`
- **Block 5 (Purple):** `#8B5CF6`
- **Block 6 (Cyan):** `#06B6D4`
- **Block 7 (Pink):** `#EC4899`

## 2. Typography
- **Primary Font:** Inter (Sans-serif)
- **Headings:** Bold (700), Slate 50
- **Body:** Regular (400), Slate 400
- **Buttons/Labels:** Medium (500), Slate 50

## 3. UI Components
### Buttons
- **Primary:** Background Indigo 600, Text White, Rounded 8px, Subtle drop shadow.
- **Secondary:** Background Slate 800, Border Indigo 600, Text White, Rounded 8px.
- **Icon Buttons:** Circle, Slate 800 background, Slate 50 icon.

### Grids
- **Cell Size:** 48dp (scalable)
- **Cell Background:** Slate 800 with 10% opacity, subtle 1px border.
- **Outer Border:** 2px solid Slate 700.

## 4. Animation Guidelines
- **Placement:** 100ms scale-up (1.1x) then settle.
- **Clearing Row:** Blocks fade out and scale down (0.8x) over 200ms. Particle burst matching block color.
- **Combo Flash:** Screen shake intensity increases with combo level. Text "X2", "X3" appears in center with spring animation.

## 5. Themes
### Block Themes
1. **Classic Modern:** Semi-transparent, subtle inner glow, rounded 4px.
2. **Neon/Cyber:** Glowing 2px border, dark core, sharp corners.
3. **Candy:** High gloss highlight, 12px rounded corners, vibrant pastels.
4. **Ocean:** Soft blues/teals, liquid-like reflections.
5. **Minimalist:** Flat colors, no borders, 2px rounded corners.

### Grid Skins
1. **Slate (Default):** Subtle Slate 900 grid.
2. **Midnight:** Pure black with glowing grid lines.
3. **Wood:** Warm brown textures with etched grid lines.
4. **Steel:** Brushed aluminum with dark recessed lines.
5. **Nebula:** Deep space background with a soft purple/blue glow behind the grid.
